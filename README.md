# Trino Casino Landing Set

Current rebuild uses React + Vite + Tailwind and starts with three standalone Polish landing pages:

- `PL/pl-01`
- `PL/pl-02`
- `PL/pl-03`

Each page has its own entry URL. `PL/pl-03` is the white mobile-first variation using the Apex phone/slots hero reference.

Run locally with:

```bash
npm install
npm run dev
```
